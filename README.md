# Meal Recipes

A Jekyll-powered recipe collection featuring three delicious meal recipes. This site is deployed using GitHub Pages.

## Recipes Included

- **Spaghetti Carbonara** - Classic Italian pasta with creamy egg and cheese sauce
- **Chicken Tikka Masala** - Flavorful Indian-inspired curry with tender chicken
- **Classic Caesar Salad** - Fresh romaine with homemade dressing and croutons

## Local Development

To run this site locally:

```bash
# Install dependencies
bundle install

# Build the site
bundle exec jekyll build

# Serve the site locally
bundle exec jekyll serve
```

Then visit `http://localhost:4000` in your browser.

## Deployment

This site is configured for GitHub Pages deployment. Simply push to the repository and GitHub Pages will automatically build and deploy the site.
